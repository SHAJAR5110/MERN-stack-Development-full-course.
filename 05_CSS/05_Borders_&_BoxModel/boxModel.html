CSS Box Model Examples

Example 1:
Border surrounds padding, which surrounds the content.
Content inside Padding inside Border.
Example 2:
Thick dashed border with padding makes space between border and content.
Example 3:
More padding increases space inside the box, making content look centered.
Example 4:
Margin adds space outside the border, separating boxes from each other.
Example 5:
Different padding on left and right makes the box wider horizontally.

Summary: The box model includes content, padding, border, and margin. Each part affects the size and spacing of elements.
