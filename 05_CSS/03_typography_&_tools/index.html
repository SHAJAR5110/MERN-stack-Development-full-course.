Color and Fonts

CSS allows you to style text with different colors and fonts. You can use the color property to change the text color and the font-family property to change the font.

Example

This is a sample text with blue color and Arial font.

CSS Code

        
            body {
                color: black; /* Default text color */
                font-family: 'Times New Roman', Times, serif; /* Default font */
            }
            .blue-text {
                color: blue; /* Blue text color */
                font-family: Arial, sans-serif; /* Arial font */
            }
