CSS Position Property Examples

1. position: static (default)

This box uses position: static. It follows the normal document flow.

2. position: relative

This box uses position: relative. It is shifted 40px right and 10px down from its normal position.

3. position: absolute

This box uses position: absolute. It is positioned 30px from the top and right of the container.

4. position: fixed

This box uses position: fixed. It stays at the bottom-right of the viewport even when scrolling.

5. position: sticky

This box uses position: sticky. It sticks to the top of the container when you scroll.
